# 🧩 Ultimate Puzzle Collection

A modern, high-performance collection of classic puzzle games built with React, Vite, and Tailwind CSS. Featuring a premium design with smooth animations and intuitive controls.

![Games Overview](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## 🎮 Included Games

- **15 Puzzle**: The classic sliding tile puzzle. Now with multi-tile sliding support!
- **Memory Match**: A sleek memory challenge with beautiful icons.
- **Lights Out**: Strategy game where you try to turn off all the lights.
- **2048**: Smooth-as-silk implementation of the viral tile-merging game.
- **Minesweeper**: A modern take on the classic mine-clearing game.
- **Word Guess**: Wordle-style challenge with unique word lists.
- **Sudoku**: Clean, minimal Sudoku with difficulty levels.
- **Code Breaker**: Mastermind-style deduction game.
- **Tower of Hanoi**: The classic mathematical puzzle.
- **Nonogram**: Logic-based grid filling puzzle.
- **Whack-a-Bug**: Fast-paced action game for developers.
- **Sassy Simon**: A memory game that judges your every move.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/drdhavaltrivedi/puzzle-collection.git
   cd puzzle-collection
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

The app will be available at `http://localhost:3000`.

## 📦 Deployment

### Deploy to Vercel

The easiest way to deploy this project is to use the [Vercel Platform](https://vercel.com/new).

1. Push your code to a GitHub repository.
2. Go to Vercel and import the repository.
3. Vercel will automatically detect the Vite configuration and deploy your app.

Alternatively, you can deploy using the Vercel CLI:
```bash
npm install -g vercel
vercel
```
